Geek Bazaar   (December 22, 2017 - January 11, 2018)

A simple web interface for sellers and buyers of board games to connect, communicate, and buy/sell/exchange board games! Built using VueJS for the client font-end and Express / NodeJS for the RESTful API server. Vuex was used for centralized state-management with Vue-router for end-point routing. PostgreSQL database with KnexJS as the SQL query builder. Data on board games is pulled from the boardgamegeek API.

Users are able to a.register/login/logout with authenticated credentials, b. explore the current inventory of games posted for sale on the Geek Bazaar inventory, c. select multiple games, adding them to the purchase cart, d. review the final order before checking out, and d. connect with the buyer/seller, to finalize the transaction and settle any disputes.
